## Project Description
This project is developed as part of the Digital Literacy course. It demonstrates the understanding and application of essential digital tools and platforms used for productivity, communication, and professional development.

The project focuses on building digital skills such as creating content, collaborating online, and maintaining digital safety.

---

## Objectives
- To understand the basics of digital literacy  
- To learn and use tools like Canva, Google Workspace, and GitHub  
- To build a digital portfolio  
- To understand email etiquette and online communication  
- To gain awareness about cybersecurity
